# Margin

A personal vocabulary-capture app: catch a word you don't know while reading, look it up,
and keep it — tagged to the book you found it in.

Testing this with family/friends? See [`guide.html`](guide.html) — the install guide, live
at [debashis9.github.io/vocab_capture/guide.html](https://debashis9.github.io/vocab_capture/guide.html).
